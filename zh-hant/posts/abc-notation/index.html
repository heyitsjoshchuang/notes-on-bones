Abc Notation

2022-08-17
2022-08-17
7分鐘閱讀

In this post, we will generate sheet music for bones using abc notation , and incorporate it into your everyday note taking by leveraging a powerful note taking app, HackMD. .

Sheet music from text

If you’re reading this, you probably already know how to write sheet music and have your own software of choice to do the job on your device. So why abc? abc notation isn’t necessarily a competitor to those software. Rather than being a tool dedicated to laying out professional looking scores, I see it as a solution for inserting sheet music into regular text files.

The abc notation system, being text-based, sits nicely within any document as long as it’s in a simple text format such as .txt or .md. So if you already have notes that are not related to music, you can easily keep your music related notes in the same place, rather than having to reach out for a dedicated score editing program to write things down. Not to mention that you will no longer be compromised on your note taking: no more jamming words into blank spots of your score!

How to write in abc

Abc notation is not the easiest to read, I’ll have to admit, but conversely it’s pretty straightforward to write, which is a fair tradeoff since it’s the generated score that we’ll be reading at the end anyways. The bulk of the syntax is pretty straight forward, and you’ll do just fine figuring things out as you come across them. The information fields, however, can be a bit confusing, so we’ll go through some use cases together. Feel free to come back to this section at any point if you need to copy and paste some stuff.

Just for the records: we are using abc notation version 2.1.

Title, meter, and unit note length

The information fields are uppercase letters followed by a colon. The most common ones only need to be written out once at the beginning; others may appear multiple times at different places. Here is an example:

% A single percent sign indicates a comment
% You can write anything here without messing up the syntax!
T: Twinkle twinkle little star % Title
M: 4/4                         % Meter
L: 1/4                         % Unit note length (?)
C C G G|A A G2|F F E E|D D C2|]

The above example generates the following score:

T: Twinkle twinkle little star M: 4/4 L: 1/4 C C G G|A A G2|F F E E|D D C2|]

The T and M fields are self-explanatory, but what does the L field (unit note length) do? Notice that the note names C C G G correspond to the first 4 quarter notes. They are quarter notes because L is set to 1/4. If L is set to, say, 1/8, then each note name correspond to an eighth note, as shown by the next example:

M: 3/4
L: 1/8
G2 CD EF|G2 C2 C2|]

…generates this:

M: 3/4 L: 1/8 G2 CD EF|G2 C2 C2|]

The 2 after a note name indicates that the note value shoule be double that of the unit note length. So if L is 1/8, C is an eighth note, C2 is a quarter note, C/2 is a sixteenth note, C3/2 is a dotted eighth note, etc.

Clef and staff line count

The K (key) field determines the key of the tune, which isn’t relavent for percussion. But it also determines the clef and the number of staff lines. To notate bones music, we probably don’t need 5 staff lines, if at all. I prefer leaving 1 staff line, but you can get rid of all of them if you want:

M: 4/4
L: 1/4
K: perc stafflines=1 % Use percussion clef, and use 1 staff line
%
% You can still plop notes up and down as if there are still 5 staff lines
C D E F|G A B c|]
M: 4/4 L: 1/4 K: perc stafflines=1 % Use percussion clef, and use 1 staff line % % You can still plop note up and down as if there are still 5 staff lines C D E F|G A B c|]
M: 4/4
L: 1/8
K: perc stafflines=0 % Use percussion clef, and use 0 staff lines
B2 B2 B2 B2|BB BB BB B2|]
M: 4/4 L: 1/8 K: perc stafflines=0 % Use percussion clef, and use 0 staff lines B2 B2 B2 B2|BB BB BB B2|]

Voices

For two-handed bones playing, use the V field to denote different voices. In our use case, we’ll use a special %%score directive (For beginners, don’t think of it as a comment since it has its functionality.) to group voices: In the following example, the 2 voices 1 and 2 are grouped together, and therefore assigned to the same staff. Note that the %%score directive is written before the voices are declared. Kinda weird.

Then we can use square brackets [] to write for different voices:

M: 4/4
L: 1/4
K: perc stafflines=1
%%score (1 2)             % Voices 1 and 2 should be on the same staff
V: 1                      % Declares voice 1
V: 2                      % Declares voice 2
[V: 1] c x           x x| % This belongs to voice 1
[V: 2] A (3A/2A/2A/2 A A| % This belongs to voice 2
[V: 1] x c c2|]           % This belongs to voice 1
[V: 2] z A A2|]           % This belongs to voice 2
M: 4/4 L: 1/4 K: perc stafflines=1 %%score (1 2) V: 1 V: 2 [V: 1] c x x x| [V: 2] A (3A/2A/2A/2 A A| [V: 1] x c c2|] [V: 2] z A A2|]

z and x are rests and hidden rests, respectively, and they follow the same rule as regular notes.

…and that’s about all of the information fields you need to know. The rest of the sheet music elements: ties, dynamics, accents, etc. are quite self-explanatory. If you’re already sold on abc notation, spend a few minutes reading the cheatsheet and you should be good to go!

Actually seeing the sheet music

Okay, so far you’ve taken my word for it that this abc notation thing generates sheet music. The next step is to find a way to actually do so for yourself.

Quick solution

If you just want to try out writing your own music in abc notation, you can go to https://editor.drawthedots.com/ , a convenient online tool for rendering abc notation. Since the sheet music is rendered as an SVG in the browser, you can also quickly get a browser extension (e.g. SVG Export for Google Chrome) to download that SVG from the webpage. For long scores, you can also use its “Print” output feature to generate a PDF of your sheet music.

HackMD

If you are looking for a long-term note taking solution, then I highly recommend using HackMD , an online collaborative Markdown-based note taking app that supports abc rendering. Type out your notes as you would normally, insert your abc notation, and see it generated real-time.

Markdown, often denoted by the .md file extension, is a elegant way of formatting pretty much any type of document, far beyond just personal notes. If you are new to markdown, The Markdown Guide has a great cheatsheet for you, though the only syntax that you absolutely need to know are headings and codefences:

# This is a level 1 heading, or h1
## h2. Headings have a larger font size and more font weight
### h3. Use headings to organize content into hierarchies!

This is a regular paragraph. The following is a codefence.
"abc" indicates that the programming language (or in our case,
notation system) is abc. Text wrapped inside the codefence
will be interpereted as abc notation.

```abc
M: 3/4
L: 1/8
G2 CD EF|G2 C2 C2|]
```

Now you have everything you need to begin note taking in abc notation and markdown. Sign in to HackMD with any Google account and start accumulating your music knoledge base :)

Side note: abcjs

abcjs is a Javascript library by Paul Rosen and community members that converts abc notation to sheet music in the browser. The sheet music rendering on this site and countless others is achieved by this very library, so big thanks to the authors! If you are looking to insert sheet music into your own website, give abcjs and abc notation a shot.
